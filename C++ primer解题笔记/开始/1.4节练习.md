### 练习1.9:

### 解：

```cpp
#include <iostream>
using namespace std;

int main(){
    int n = 50;
    int sum = 0;
    while(n <= 100){
        sum += n;
        n++;
    }
    cout << sum << endl;
    return 0;
}
```
