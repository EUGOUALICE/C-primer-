### 练习1.6，解：

### 不合法，每行换行后面不应该添加分号。



### 出题思路：

### 让读者体会输出语句分段容易出现的错误。



### 官方解答：

### 不合法。

### 前两行的末尾有分号，表示语句结束，第2、3两行为两条新的语句。而这两条语句在“<< ”之前缺少了输出流，应在“ << ”之前加上"std::cout" ， 即得到正确的程序

```cpp

int main(){
    cout << "hello" << "\t" <<"men" << "\t" <<"you are";
     cout << "a*b" <<"\t"<< "a" <<"\t"<< "good" <<"\t"<< "boy" << endl;
    return 0;
}
```
